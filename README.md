# DatePicker
移动端滚动式日期控件，带平滑滚动效果，超小体积，原生js编写，不依赖任何第三方文件，绝不含任何添加剂。

## 快速开始
1、引入文件
```HTML
<html>
	<head>
		...
		<link rel="stylesheet" href="path/to/datepicker.css" />
	</head>
	<body>
		...
		<script src="path/to/datepicker.js"></script>
	</body>
</html>

```
2、使用
```HTML
<input type="text" role="datepicker" data-format="yyyy-MM-dd" readonly />
```
